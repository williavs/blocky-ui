# Installation

Follow these steps to add Blocky to your project.

## Prerequisites

Before you begin, make sure you have:

- Node.js 14 or later
- npm or yarn
- React 17 or later

## Setup Steps

### 1. Install Dependencies

First, install the required dependencies:

```bash
npm install tailwindcss @tailwindcss/typography clsx tailwind-merge lucide-react
```

### 2. Add Tailwind CSS Configuration

Create or update your `tailwind.config.js`:

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {
      fontFamily: {
        sans: ['var(--font-sans)'],
        'space-grotesk': ['var(--font-space)'],
      },
      colors: {
        background: 'var(--background)',
        foreground: 'var(--foreground)',
        card: 'var(--card)',
        'card-foreground': 'var(--card-foreground)',
        popover: 'var(--popover)',
        'popover-foreground': 'var(--popover-foreground)',
        primary: 'var(--primary)',
        'primary-light': 'var(--primary-light)',
        'primary-dark': 'var(--primary-dark)',
        muted: 'var(--muted)',
        'muted-foreground': 'var(--muted-foreground)',
        accent: 'var(--accent)',
        'accent-foreground': 'var(--accent-foreground)',
        destructive: 'var(--destructive)',
        'destructive-foreground': 'var(--destructive-foreground)',
        border: 'var(--border)',
        ring: 'var(--ring)',
      },
    },
  },
  plugins: [
    require('@tailwindcss/typography'),
  ],
}
```

### 3. Add CSS Variables

Add these CSS variables to your root CSS file (e.g., `index.css`):

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

:root {
  --font-sans: 'DM Sans', system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue',
    Arial, 'Noto Sans', sans-serif;
  --font-space: 'Space Grotesk', system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
    'Helvetica Neue', Arial, 'Noto Sans', sans-serif;

  /* Light Theme Variables */
  --primary: rgb(187, 247, 208);
  --primary-light: rgb(240, 253, 244);
  --primary-dark: rgb(20, 83, 45);
  --background: rgb(240, 253, 244);
  --foreground: rgb(20, 83, 45);
  --card: white;
  --card-foreground: rgb(20, 83, 45);
  --border: black;
  --ring: rgb(134, 239, 172);
}

[data-theme="dark"] {
  --primary: rgb(134, 239, 172);
  --primary-light: rgb(20, 83, 45);
  --primary-dark: rgb(240, 253, 244);
  --background: rgb(17, 24, 39);
  --foreground: rgb(240, 253, 244);
  --card: rgb(24, 33, 47);
  --card-foreground: rgb(240, 253, 244);
  --border: rgb(134, 239, 172);
  --ring: rgb(134, 239, 172);
}
```

### 4. Add Font Dependencies

Add the required fonts to your HTML:

```html
<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&family=DM+Sans:wght@400;500;700&display=swap" rel="stylesheet">
```

## Verifying Installation

To verify that everything is working, try adding a simple button component:

<div className="not-prose p-4 border-2 border-black rounded-lg bg-white shadow-[4px_4px_0px_0px_rgba(0,0,0,1)]">
  <Button>Hello Blocky!</Button>
</div>
