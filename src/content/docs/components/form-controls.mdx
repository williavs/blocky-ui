# Form Controls

A collection of form control components with our signature blocky style. Each component is designed to be accessible and easy to use.

## Import

```tsx
import { Input, TextArea, Select, Checkbox, Radio, Switch } from '@blocky/react';
```

## Usage

```tsx
<Input placeholder="Enter your name" />
<TextArea placeholder="Enter your message" />
<Select options={[{ value: 'option1', label: 'Option 1' }]} />
<Checkbox label="Accept terms" />
<Radio label="Select this option" />
<Switch label="Enable feature" />
```
